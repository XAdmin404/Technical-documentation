Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming. Due to its vast ecosystem of libraries, Python is widely used in web development, data science, automation, and more.

What You Should Already Know

To follow this documentation, you should have a basic understanding of:

  • General programming concepts (variables, loops, functions)
  • How to use a text editor or an IDE
  • Basic command-line operations

Python and Other Languages

Python is dynamically typed, meaning you don't have to declare variable types explicitly. Unlike Java or C++, Python uses indentation instead of curly braces for code blocks, making it more readable and concise.

Hello World

The classic first program in Python:


print("Hello, World!")

This prints "Hello, World!" to the console.

Variables

Variables in Python do not require explicit type declaration. You can assign values directly:


name = "Alice"
age = 25
print(name, age)

Output


Alice 25

Data Types

Python has several built-in data types, including:

  • int (e.g., 10)
  • float (e.g., 3.14)
  • str (e.g., 'hello')
  • bool (e.g., True, False)

If Statement

Conditional statements in Python use indentation for defining blocks:


x = 10
if x > 5:
    print("x is greater than 5")

While Loop

A while loop repeats a block of code while a condition remains true:


count = 0
while count < 5:
    print(count)
    count += 1

For Loop

Used for iterating over sequences like lists or strings:


for i in range(5):
    print(i)

Functions

Functions in Python are defined using def:


def greet(name):
    return f"Hello, {name}!"
print(greet("Alice"))

Conclusion

Python is a versatile and beginner-friendly language. With its clean syntax and powerful libraries, it is an excellent choice for various programming tasks. Keep exploring and practicing!
